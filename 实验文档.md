# 实验流程

  胸部 CT
    ↓
  胸腺中心定位 CoM        当前已有 locator
    ↓
  围绕 CoM 裁剪 ROI       当前已有 patch generation
    ↓
  3D ResNet50 提取特征    当前已有 SSL backbone，输出4096维特征
    ↓
  预测模型               当前已有线性分类头雏形
    ↓
  胸腺健康评分           当前只做二分类/评分标签验证，可靠性还不足

# locator

  50CT main.nii
    ↓
  读取 mark.nii 中的点标注
    ↓
  把点标注转成 3D 高斯热图
    ↓
  训练 3D Heatmap U-Net
    ↓
  推理时输出热图
    ↓
  取热图最大值作为预测 CoM

# baseline


1. Dummy baseline
 特征：无
 模型：most_frequent / stratified
 作用：最低参考线

2. HU statistics baseline
 特征：patch 强度统计
 包括 mean, std, min, max, percentile, HU区间比例
 模型：LogisticRegression / Ridge / LinearSVM
 作用：判断简单灰度统计是否已经足够

3. Random ResNet baseline
 特征：随机初始化 3D ResNet50 提取 4096维
 backbone：冻结
 模型：同样的浅层分类头
 作用：判断性能是否来自网络结构本身，而不是 SSL 预训练

4. SSL ResNet baseline
 特征：TCIA SSL 预训练 3D ResNet50 提取 4096维
 backbone：冻结
 模型：同样的浅层分类头
 作用：主实验


# 分类头配置
  不要用神经网络头，50 例太少。建议每组都统一跑：

  DummyClassifier
  RidgeClassifier
  LogisticRegression
  LinearSVM
  SelectKBest(k=5,10,20,50) + LogisticRegression
  PCA(n=5,10,20) + LogisticRegression

  其中推荐主报告：

  SelectKBest + LogisticRegression
  RidgeClassifier
  PCA + LogisticRegression

# 交叉验证配置

  outer_cv = RepeatedStratifiedKFold(
      n_splits=5,
      n_repeats=50,
      random_state=42
  )

  如果要调参，必须放在内层：

  inner_cv = StratifiedKFold(
      n_splits=3,
      shuffle=True,
      random_state=42
  )

  可调参数：

  LogisticRegression C: [0.01, 0.1, 1, 10]
  SelectKBest k: [5, 10, 20, 50]
  PCA n_components: [5, 10, 20]
  SVM C: [0.01, 0.1, 1]
  Ridge alpha: [0.1, 1, 10, 100]
