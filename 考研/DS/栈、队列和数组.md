# 栈
* 栈顶指针指向栈顶的下一个元素（教材一般定义）
*  n个元素进栈时，出栈元素不同排列为$\displaystyle \frac{1}{n+1} \binom{n}{2n}$,Catlan公式
* 链栈的入栈出栈操作都在表头进行
* 应用
	* 数制转换（余数进栈然后出栈）
	* 括号匹配的检验（匹配则消解，否则进栈） 
	* 表达式求值
		* 算术表达式
		* 中缀转后缀
			* 遇到操作数加入后缀表达式
			* 遇到（），遇到“（”入栈，遇到“）”，将（之前的都加入后缀表达式，删除（
			* 遇到运算符，将高于等于它的操作符加入后缀表达式
		* 后缀表达式求值
			* 操作数入栈
			* 运算符，弹出两个操作数Y,X,==运算 X  op Y==

# 特殊矩阵的压缩存储

* 对称矩阵
	* $\displaystyle \frac{i(i-1)}{2}+j-1,\quad i\geq j$   下三角区域和对角线
	* $\displaystyle \frac{j(j-1)}{2}+i-1, \quad i<j$
* 三角矩阵
	* $\displaystyle \frac{i(i-1)}{2}+j-1$
	* $\displaystyle \frac{n(n+2)}{2}$  常数项
* 三对角矩阵
	* 2i+j-3
	* 行加一，加一，列加一，加二
* 稀疏矩阵
	* 三元组
	* 行逻辑连接的顺序表
	* 十字链表
