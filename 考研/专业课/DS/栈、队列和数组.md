# 栈
* 栈顶指针指向栈顶的下一个元素（教材一般定义）

> [!solution]+ 卡特兰数
> $\displaystyle \frac{1}{n+1} \binom{n}{2n}$
>使用条件
>1.n个元素进行栈操作（入栈序列确定），有多少种合法出栈序列
>2.先序序列含n个不同的元素，能够确定有多少种不同的二叉树
>3.先序序列有n个不同元素，对应的不同形状的二叉树的个数
>
* ==链栈的入栈出栈操作都在表头进行==
* 应用
	* 数制转换（余数进栈然后出栈）
	* 括号匹配的检验（匹配则消解，否则进栈） 
	* 表达式求值
		* 算术表达式
		* 中缀转后缀
			* 遇到操作数加入后缀表达式
			* 遇到（），遇到“（”入栈，遇到“）”，将（之前的都加入后缀表达式，删除（
			* 遇到运算符，将高于等于它的操作符加入后缀表达式
		* 后缀表达式求值
			* 操作数入栈
			* 运算符，弹出两个操作数Y,X,==运算 X  op Y==

# 队列
**链队列**
* 带头结点，头指针指向头结点
* 判空：头指针尾指针均指向头结点，->next=Null
**循环队列**：
* ==front指向第一个元素，rear指向最后一个的下一个位置==
	* 初始值：rear == front = 0
	* 判满：（rear+1）%n=front(牺牲一个单元，最后一个单元不放东西)
	* 判空：rear=front
	* 元素个数：（rear-front+n）%n
* front指向第一个元素，rear指向最后一个元素
	* 初始值：front=0；rear指向队尾
	* 队空：（rear+1）%n= front

# 特殊矩阵的压缩存储

* 对称矩阵
	* $\displaystyle \frac{i(i-1)}{2}+j-1,\quad i\geq j$   下三角区域和对角线
	* $\displaystyle \frac{j(j-1)}{2}+i-1, \quad i<j$
* 三角矩阵
	* $\displaystyle \frac{i(i-1)}{2}+j-1$
	* $\displaystyle \frac{n(n+2)}{2}$  常数项
* 三对角矩阵
	* 2i+j-3
	* 行加一，加一，列加一，加二
* 稀疏矩阵
	* 三元组
	* 行逻辑连接的顺序表
	* 十字链表
