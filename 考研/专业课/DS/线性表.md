# 一、基本概念
	具有相同数据类型的n个数据元素的有限序列

# 二、实现
### 顺序存储
	顺序表：用一组地址连续的存储单元依次存储线性表，逻辑顺序与物理顺序也相同
优点：
	随机访问
	存储密度高
缺点：
	插入平均移动n/2
	删除平均移动（n-1）/2
	顺序查找，(n+1)/2
	需要分配连续存储空间不灵活
