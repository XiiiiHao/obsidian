# 文件

### 文件的逻辑结构
1. 顺序文件
2. 索引文件
	索引号+长度m+指针ptr,平均查找$\displaystyle \frac{n+1}{2}次$
3. 索引顺序文件
	$\displaystyle \sqrt{ n }次$

### 文件的物理结构
1. 连续分配
	1. 顺序访问和直接访问，顺序访问容易且速度快
	2. 缺点：外部碎片，不能动态增长，删除插入需要移动相邻记录
2. 链接分配
	1. 隐式链接：file+start+end
	2. 显式链接：FAT,盘块号+下一块
3. 索引分配
4. 多级索引
5. inode,混合索引

### 文件共享
1. 硬链接
	count等于指向当前inode的数量
2. 软链接
	创建一个新的inode指向要共享的文件，新的inode的count=1

### 磁盘管理
1. 空闲表法：适合连续分配
	序号+第一个空闲盘号+空闲盘块数
2. 空闲链表法
	空闲盘块链
	空闲盘区链
3. 位示图
	默认从1开始编号
4. 成组链接法
