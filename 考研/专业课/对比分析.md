# 计网
### 数据链路层
> [!key]+ CSMA/CD与CSMA/CA的区别
>1.CD可以检测冲突，但无法避免；CA在发送数据的同时不能检测信道上有无冲突
>2.传输介质不同；CD用于总线型以太网，CA用于无线局域网
>3.检测方式不同；cD检测电缆中的电压变化，CA采用能量检测

### 网络层
**数据报服务与虚电路服务的比较**：

|           | 数据报服务               | 虚电路服务                 |
| --------- | ------------------- | --------------------- |
| 连接的建立     | 不需要                 | 需要                    |
| 目的地址      | 每个分组包含完整的目的地址与源地址   | 仅在建立连接的时候使用，然后就使用虚电路号 |
| 路由选择      | 每个分组独立的进行路由选择和转发    | 同一虚电路的分组按同一路由转发       |
| 分组有序性     | 不保证有序到达             | 保证有序到达                |
| 可靠性       | 不可靠尽最大努力交付，可靠性靠用户主机 | 可靠性由网络保证              |
| 对网络故障的适应性 |                     |                       |
| 差错处理和流量控制 | 用户主机进行流量控制，不保证可靠    | 可由分组交换网负责，也可由用户主机负责   |
