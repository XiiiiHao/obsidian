# 通信基础
基本概念：
	码元传输速率：波特率or调制速率，每秒传送的码元数，单位Baud
	信息传输速率：比特率
==奈氏准则==：无噪声最大数据传输速率
	$\displaystyle c=2W \log_{2}V$,V是有多少种码元
==香农定理==：带宽受限且有高斯噪声干扰
	$\displaystyle c=W\log_{2}\left( 1+\frac{S}{N} \right)$
	信噪比无单位时，$\displaystyle \frac{S}{N}=信噪比$
	信噪比以分贝为单位时，信噪比=$\displaystyle 10\log_{10}\left( \frac{S}{N} \right)$
### 编码与调制
1. 数字数据编码为数字信号
	1. 归零（RZ）编码：高电平为1,低电平为0（或相反），==码元中间时刻均归零==
	2. 非归零编码：不归零，高1低0
	3. 反向归零编码：电平跳变表示0,电平不变表示1
	4. 曼彻斯特编码：每个码元中间发生电平跳变，下跳1上跳0or相反；标准以太网在使用
	5. 差分曼彻斯特编码：每个码元中间发生电平跳变，但看码元开始时，无跳变表示1,有跳变表示0
2. 模拟数据编码为数字信号
	1. 采样
	2. 量化
	3. 编码
3. 数字数据调制为模拟信号
	1. 调幅（AM）
	2. 调频（FM）
	3. 调相（PM）
	4. 正交幅度调制QAM：$\displaystyle R=B\log_{2}mn$,m为相位个数，n为幅度个数
4. 模拟调制为模拟：频分复用
### 传输介质
物理层的接口特性：
	机械特性：接口形状，尺寸。引脚数量
	电气特性：电压范围，传输速率，距离限制
	功能特性：电压的意义，每条线的功能
	过程特性：时序关系
