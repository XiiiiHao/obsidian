# 概率论
* 判定分布函数（连续变量）
	充要条件：单调不减，右连续，F(+无穷) = 0, F(-无穷) = 1
	常见形式：$\displaystyle \sum^{k}_{i=1} \lambda_{i}F_{i}(x)及其连续形式(即F(x)在某个区间上的均值)$，均方根$\displaystyle \sqrt{ F_{1}(x)F_{2}(x) }$, $\displaystyle [F(x)]^{n}, 1-[1-F(x)]^{n}$ ^4025d1
* 判定分布律（离散变量）
	大于等于0,和为1
* 判定概率密度
	大于等于0,和为1
	常见：线性组合连续形式，$\displaystyle \frac{2}{n}\sum^{n}_{i=1}F_{i}(x)f_{i}(x)$, $\displaystyle F_{1}(x)F_{2}(x)F_{n}(x)的导数$，$\displaystyle n[F(x)]^{n-1}f(x),n[1-F(x)]^{n-1}f(x)$
	==均方根不可以作为密度函数==
* 独立性与相关性判断
	* $\displaystyle F(x,y)=F_{X}(x) \cdot F_{Y}(y)$证明不独立就找一个不满足的
	* 二维离散：$\displaystyle p_{ij}=p_{i}p_{j}$; 二维连续:$\displaystyle f(x,y)=f_{X}(x)f_{Y}(y)$
	* f(x,y)的非零区域不是矩形，则X与Y不独立
	* $\displaystyle \rho_{XY}=0 \leftrightarrows Cov(X,Y)=0 \leftleftarrows E(XY)=E(x)E(Y) \leftrightarrows D(X \pm Y)=DX+DY$
	* 步骤：先算cov,不为0则相关，则不独立，为0不相关，通过分布判断是否独立
