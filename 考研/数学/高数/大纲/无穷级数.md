# 常数项级数的概念与性质
性质：
1. 收敛级数的线性组合也收敛
2. 改变有限项不改变敛散性
3. 收敛级数加括号还收敛，反之未必；加了括号发散，原级数必发散
4. 收敛的必要条件:$\displaystyle \lim_{ n \to \infty }u_{n}=0$

# 级数敛散性的判别方法

### 正项级数
1. 收敛原则：部分和有界（充要条件）
2. 比较判别法和极限形式
	大敛小敛，小散大散
> [!theorem]+ 正项级数
> $\displaystyle \sum a_{n}收敛，则\sum a^{2}_{n}、\sum a^{3}_{n}、\sum a_{2n} 、\sum a_{2n+1}收敛$
> 两个正项级数收敛，乘积也收敛
3. 比值判别法
	对于正项级数$\displaystyle \sum u_{n},如果\lim_{ n \to \infty } \frac{u_{n+1}}{u_{n}}= \rho$
	$\displaystyle \rho<1，收敛，\rho>1，发散$
4. 根值判别法
	对于正项级数$\displaystyle \sum u_{n},如果\lim_{ n \to \infty } \sqrt[n]{u_{n}}= \rho$
	$\displaystyle \rho<1，收敛，\rho>1，发散$
5. 积分判别法
	$\displaystyle 正项级数 \sum u_{n},若存在[1,\infty)单调减少的非负连续函数f(x),使得u_{n}=f(x),则级数与反常积分\int^{\infty}_{1}f(x)的敛散性相同$

### 交错级数及其敛散性判别
莱布尼兹判别法
	单调不增，极限为0,则收敛
