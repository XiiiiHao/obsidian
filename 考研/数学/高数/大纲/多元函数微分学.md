### 可微
$$
\Delta z = A \Delta x + B \Delta y + o(\rho),\rho= \sqrt{ (\Delta x)^{2}+ (\Delta y)^{2} }
$$
必要条件：偏导存在
充分条件：偏导数存在且连续
> [!solution]+ 判断偏导数是否连续
>1.定义法求$\displaystyle f^{'}_{y}(x_{0},y_{0}),f^{'}_{y}(x_{0},y_{0})$
>2.公式法求偏导数
>3.计算公式法的极限

> [!solution]+ 可微的判别步骤
>1.$\displaystyle \Delta z=f(x_{0}+\Delta x,y_{0}+\Delta y)-f(x_{0},y_{0})$
>2.写出线性增量$\displaystyle A \Delta x+B \Delta y$
>3.求极限$\displaystyle \lim_{ \begin{align} \Delta x \rightarrow 0 \\ \Delta y \rightarrow 0 \end{align} } \frac{\Delta z - A \Delta x+B \Delta y}{\sqrt{ (\Delta x)^{2}+(\Delta y)^{2} }}$
