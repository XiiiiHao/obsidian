# 程序和进程

## 进程（状态机）管理

### fork（）:
```c
#include <unistd,h>
	pid_t fork(void); 
```
* 子进程返回0，父进程返回子进程的pid,fork失败返回-1


### exec()函数家族:
* l（list）:参数以可变参数列表传递
* v（vector）：参数以字符串数组（`char *const argv[]`）传递
* `p`（PATH）：自动在 `PATH` 环境变量指定的路径中搜索可执行文件
* `e`（Environment）：允许自定义环境变量（通过 `char *const envp[]` 参数）
exceve():
```c
#include <unistd.h>
	int execve(const char *pathname, char *const _Nullable argv[],
			  char *const _Nullable envp[]);
```
* argv与envp均以NULL结尾
* 成功不返回，失败返回-1,errno来报错


### wait()与waitpid()
```c
#include <sys/wait.h>
   pid_t wait(int *_Nullable wstatus);
   pid_t waitpid(pid_t pid, int *_Nullable wstatus, int options);
```
* wait()只等待第一个子进程结束，总是阻塞等待，成功：返回被终止的子进程的pid,失败：返回-1
* waitpid()同时支持阻塞（options=0）与非阻塞模式(options=WNOHANG)
